# FlutterNFT
FlutterNFT是一个基于 Flutter + Solidity 开发的开源去中心化应用（DApp），支持 数字藏品（NFT）发行、交易与展示。
跨平台支持 iOS / Android / Web，助力开发者快速搭建属于自己的 去中心化数字藏品市场。

#### 功能特性

* 🔐 钱包集成：支持连接 MetaMask 等 Web3 钱包

* 🎨 数字藏品管理：铸造 (Mint)、上架、转让、销毁

* 🏪 交易市场：用户可自由浏览、购买、出售 NFT

* 📱 跨平台支持：Flutter 构建，支持 iOS / Android / Web

* 🌍 去中心化：所有数据与交易在区块链上执行，透明可信
